# Statistical Analysis

## Hypothesis Tests

**H0**: Voting accuracy = Single sample accuracy
**H1**: Voting accuracy > Single sample accuracy

### k1_live

- **n**: 240
- **Voting mean**: 25.0%
- **Single mean**: 21.2%
- **t-statistic**: 2.5239
- **p-value**: 0.011605
- **Cohen's d**: 0.1629 (negligible)
- **Conclusion**: Reject H0 - Voting significantly improves accuracy

### k2_live

- **n**: 240
- **Voting mean**: 25.0%
- **Single mean**: 22.9%
- **t-statistic**: 1.3894
- **p-value**: 0.164700
- **Cohen's d**: 0.0897 (negligible)
- **Conclusion**: Fail to reject H0

### k3_live

- **n**: 240
- **Voting mean**: 22.9%
- **Single mean**: 24.2%
- **t-statistic**: -0.9042
- **p-value**: 0.365895
- **Cohen's d**: -0.0584 (negligible)
- **Conclusion**: Fail to reject H0
