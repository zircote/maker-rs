# Project Planning Template

> This repository was created from the [Project Planning Template](https://github.com/zircote/project-planning-template). Follow the steps below to generate your project planning artifacts.

## Getting Started

### 1. Define Your Project Context

**Option A: Interactive Interview (Recommended)**

Open Claude Code in this repository and run:
```text
/project-context
```

**Option B: Manual Creation**

Copy `.claude/templates/PROJECT-CONTEXT.md` to the repository root and fill in your project details:
```bash
cp .claude/templates/PROJECT-CONTEXT.md PROJECT-CONTEXT.md
```

### 2. Generate All Artifacts

```text
/project-architect Generate complete project planning artifacts based on PROJECT-CONTEXT.md
```

See [Command Guide](.claude/docs/COMMAND-GUIDE.md) for the full reference.
